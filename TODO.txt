1. 消除播放延迟
  在我电脑上未发现明显延迟，暂时不考虑 audio 标签的性能问题；
  在中间 doorbump 的位置出现可复现的问题，从上/下移动到门口时，第一次撞门无声音。
  故考虑是代码播放逻辑的问题。

  吴：每个单个audio无论是否触发播放逻辑，如果其正处于播放中，触发播放无效，需等待其播完以后再次触发。

2. 代码结构拆分
  // 新建一个 html 文件，两个 js 文件：a.js、b.js
  // 在 b.js 直接执行的代码依赖 a.js 时
  // a b 可以
  // b a 不行
  // 如果没有直接执行的，都是定义，那么顺序是无所谓的，因为 js 边执行边解释，只要执行到的时候定义好了就行，比如：
  let f = () => console.log(abc) // 此并无 abc
  f() // 此时调用会报错：Uncaught ReferenceError: abc is not defined
  let abc = 1234
  f() // 此时调用就 ok
